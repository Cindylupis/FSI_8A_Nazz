## Descripción

Morse code is well known. Can you decrypt this?

Download the file [here](https://artifacts.picoctf.net/c/79/morse_chal.wav).

Wrap your answer with picoCTF{}, put underscores in place of pauses, and use all lowercase.
## Solución

**Solución 1
```
WH47 H47H 90D WR0U9H7
Flag :picoCTF{wh47_h47h_90d_w20u9h7}

```

#### NOTAS
El mensaje original es **"WHAT HATH GOD WROUGHT"** (¿Qué ha hecho Dios?). Fue el **primer mensaje transmitido por telégrafo en la historia** por Samuel Morse. El creador de este reto de picoCTF simplemente lo escribió usando _leetspeak_ (cambiando letras por números parecidos).
